Updates for Polyray 1.8a :
   - Several problems with axis aligned triangles in raw files
   - Added texture coordinates to .obj files
   - Allow vertex/texture coordinates in raw files (instead of vert/norm/tex)
   - Added DLL based texture functions
   - Currently Win32 only


If you have problems running polyray, add this to your autoexec.bat

set POLYRAY_PATH=c:\
